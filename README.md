Project Title
Expense Tracker

Project Overview
The Expense Tracker is a simple Java console application for managing personal expenses.
Users can quickly add, view, and total their expenses for better financial awareness
. The app uses an interactive menu to streamline the process while storing expense data in memory,
making it an ideal learning exercise in core Java programming and object-oriented concepts.​

Features

Add new expense entries with custom titles and amounts.
View a list of all recorded expenses.
Calculate and display the total expenses.
Simple menu-driven console interface for easy interaction.
Graceful handling of invalid menu choices.

Technologies/Tools Used

Java (JDK 8 or higher recommended)
Any IDE (such as Eclipse, IntelliJ IDEA, or NetBeans)​
Console/Terminal for running the application

Steps to Install & Run the Project

Ensure Java is installed (JDK 8+).
Open your preferred IDE and create a new Java project.
Copy the provided code into a new class file (ExpenseTracker.java).
Build the project and run ExpenseTracker.java through the IDE or terminal using java ExpenseTracker.​

Instructions for Testing

Launch the application; the console menu appears.
Use option 1 to add a new expense by entering its title and amount.
Use option 2 to view all recorded expenses.
Use option 3 to display the total amount spent.
Use option 4 to exit the application.
Try entering invalid choices to view error handling in action.

This project helps users practice basic Java, including collections, classes,
and console input/output, while developing a useful financial utility.
